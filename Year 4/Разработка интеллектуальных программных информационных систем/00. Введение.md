Система - упорядоченная совокупность материальных объектов, объединённых связями, для достижения цели

## Свойства систем
- Система обладает новыми свойствами, которыми не обладают отдельные элементы
- Обладает свойствами оптимальности
- Создаются для решения определённых задач
# Описание системы
## Классификация
- По виду объекта (технические, биологические, социальные, тд)
- По характеру поведения (детерминированные, вероятностные, игровые)
- По типу целеустремленности (открытые, закрытые)
- По сложности структуры и поведения
- По виду научного направления
- По степени организованности
# Свойства систем
- Статические
1. Целостность
2. Открытость
- Связь со средой
